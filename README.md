Semi-Realistically types whats in your keboard. Just copy what you want to type and click into a doc. It doesnt have a real way to stop other than
clicking stop in you IDE. (I used IntelliJ). It also could probably be more realistic at typeing and add changes after it has finished typeing the 
text (but the changes are pre decided before it clicking run). This is just a fun project and not intended to be used for any malicious things or 
cheating.
